# Video Editing Client Management System

A SQL + Python based DBMS project for managing freelance workflows.

## Features

- Add and manage clients
- Add and manage projects
- Record payments
- Track revisions
- Generate full project reports
- Terminal version and Tkinter GUI version
- MySQL database integration

## Technologies Used

- Python
- MySQL
- Tkinter
- MySQL Connector
- SQL

## Database Tables

- Clients
- Projects
- Payments
- Revisions
- Editors
- Project_Assignment

## SQL Features Used

- Primary Keys
- Foreign Keys
- Constraints
- Views
- Stored Procedures
- Triggers
- Aggregate Queries
- Joins

## How to Run

1. Install MySQL
2. Create the database in MySQL Workbench
3. Run the SQL table creation queries
4. Install required package:

pip install mysql-connector-python
